Hosted Domain Owner Flow

Connect domain, verify ownership, set records, and watch continuity.

This page is the current runtime-facing control plane for domain owners. It uses the real TollDNS gateway APIs for verification, continuity, registrar status, banner state, and renewal intent. If the optional hosting control plane is running, it can also build an edge/site plan.

MVP boundary: domain verification and continuity state are live. Basic record management is generated as exact repo commands until a hosted record-write API is promoted into the default gateway path.

Default Nameservers

  • ns1.tolldns.io
  • ns2.tolldns.io

Recommended Order

  1. Load overview
  2. Start TXT verification
  3. Switch nameservers
  4. Build record plan
  5. Check continuity and renewal state
