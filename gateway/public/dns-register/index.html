Register a .dns name

This UI uses the repo's current `ddns_names` flow. It can check whether the names program is deployed and configured on the selected RPC, preview the exact wallet-backed commands, and optionally execute registration if the gateway is explicitly configured for writes. Premium names mint a canonical NFT to the owner wallet; non-premium names mint a canonical NFT into program custody.

Current framework rule: browser preview is always safe. Server-side execution only works when `DDNS_NAMES_WRITE_ENABLED=1` and an admin token is supplied.
